<a id="Distance from anchor">
  <h3>Distance from anchor</h3>
</a>

To adjust the distance between the tooltip and its anchor, you can use the slotProps prop to modify the offset of the popper.

<div className="showcase">
  <div className="items">
    <ThemeWrapper>
      <TooltipDemo type="offset" />
    </ThemeWrapper>
  </div>
  <CodeBlock name="distance-anchor" code={`<CustomTooltip
  title="Add"
  slotProps={{
    popper: {
      modifiers: [
        {
          name: "offset",
          options: {
            offset: [0, -14],
          },
        },
      ],
    },
  }}
>
  <span>
    <Button variant="contained" size="medium">
      Offset
    </Button>
  </span>
</CustomTooltip>`} />
</div>

<a id="Variable width">
  <h3>Variable width</h3>
</a>

The variant prop in Tooltip allows you to control the tooltips size with predefined styles (standard or small). This makes it easier to adjust the tooltip to fit various UI requirements quickly and consistently.

<div className="showcase">
  <div className="items">
    <ThemeWrapper>
      <TooltipDemo type="variable-width" />
    </ThemeWrapper>
  </div>
  <CodeBlock name="variable-width" code={`<CustomTooltip title="Add" variant="standard">
  <span>
    <Button variant="contained" size="medium">
      standard
    </Button>
  </span>
</CustomTooltip>
<CustomTooltip title="Add" variant="small">
  <span>
    <Button variant="contained" size="medium">
      small
    </Button>
  </span>
</CustomTooltip>`} />
</div>

<a id="Follow cursor">
  <h3>Follow Cursor</h3>
</a>

Tooltips can be configured to follow the cursor, making it easier to read the content.

<div className="showcase">
  <div className="items">
    <ThemeWrapper>
      <TooltipDemo type="follow-cursor" />
    </ThemeWrapper>
  </div>
  <CodeBlock name="follow-cursor" code={`<CustomTooltip title="Follow Cursor" followCursor>
  <span>
    <Button variant="contained" size="medium">
      Button
    </Button>
  </span>
</CustomTooltip>`} />
</div>

<a id="Showing and hiding">
  <h3>Showing and Hiding</h3>
</a>

A delay in showing or hiding the tooltip can be added through the `enterDelay` and `leaveDelay` props.

<div className="showcase">
  <div className="items">
    <ThemeWrapper>
      <TooltipDemo type="show-hide" />
    </ThemeWrapper>
  </div>
  <CodeBlock name="show-hide" code={`<CustomTooltip title="Add" enterDelay={500} leaveDelay={200}>
  <span>
    <Button variant="text" size="medium">
      [500MS, 200MS]
    </Button>
  </span>
</CustomTooltip>`} />
</div>
