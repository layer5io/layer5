The Paper component provides an elevated surface for displaying content. It mimics the behavior of real-world surfaces with shadow effects, supporting Material Design's elevation system.

<a id="Elevation Example">
  <h3>Elevation Example</h3>
</a>

The `elevation` prop controls the shadow depth. Use values from 0 to 24 to create varying levels of elevation:

  <div className="items">
    <ThemeWrapper>

          Elevation 3
        </Paper>
        <Paper elevation={8} sx={{ padding: "16px" }}>
          Elevation 8
        </Paper>
      </div>
    </ThemeWrapper>
  </div>
  <CodeBlock name="elevation-example" code={`<Paper elevation={0} sx={{ padding: "16px" }}>

  Elevation 3

<Paper elevation={8} sx={{ padding: "16px" }}>

</div>

<a id="Variant Example">
  <h3>Variant Example</h3>
</a>

The Paper component supports two variants: `elevation` (default) and `outlined`. The outlined variant removes shadows and adds a border instead:

<div className="showcase">
  <div className="items">
    <ThemeWrapper>
      <div style={{ display: 'flex', gap: '16px', flexDirection: 'column', width: '100%' }}>
        <Paper variant="elevation" sx={{ padding: "16px" }}>
          Elevation Variant (Default)
        </Paper>
        <Paper variant="outlined" sx={{ padding: "16px", borderColor: "#ccc" }}>
          Outlined Variant (No shadow)
        </Paper>
      </div>
    </ThemeWrapper>
  </div>
  <CodeBlock name="variant-example" code={`<Paper variant="elevation" sx={{ padding: "16px" }}>
  Elevation Variant (Default)
</Paper>
<Paper variant="outlined" sx={{ padding: "16px", borderColor: "#ccc" }}>
  Outlined Variant (No shadow)
</Paper>`} />
</div>

<a id="Corners Example">
  <h3>Square and Rounded Corners</h3>
</a>

By default, the Paper component has rounded corners. You can make it square by setting the `square` prop to `true`.

<div className="showcase">
  <div className="items">
    <ThemeWrapper>
      <div style={{ display: 'flex', gap: '16px', flexDirection: 'column', width: '100%' }}>
        <Paper square={false} sx={{ padding: "16px" }}>
          Rounded Corners (Default)
        </Paper>
        <Paper square sx={{ padding: "16px" }}>
          Square Corners
        </Paper>
      </div>
    </ThemeWrapper>
  </div>
  <CodeBlock name="corners-example" code={`<Paper square={false} sx={{ padding: "16px" }}>
  Rounded Corners (Default)
</Paper>
<Paper square sx={{ padding: "16px" }}>
  Square Corners
</Paper>`} />
</div>
