<p>
    Service Mesh Interface provides a standard interface for service meshes on Kubernetes. SMI provides a standard interface for service meshes on Kubernetes and a basic feature set for the most common service mesh use cases. 
</p>
<p>
    Meshery is the official <Link to="/projects/service-mesh-interface-conformance">SMI Conformance Validator</Link>. Use Meshery's diagnostic tool to verify that your service mesh's behavior in an accessible and non-destructive manner.
</p>

    After you <Link to="https://layer5.io/blog/service-mesh-specifications/service-mesh-specifications-and-why-they-matter">Understand Service Mesh Specifications and Why They Matter</Link>, you can use Service Mesh Interface and Service Mesh Performance specs on your CI/CD pipelines with Meshery's GitHub Actions by <Link to="https://layer5.io/blog/service-mesh-specifications/pipelining-service-mesh-specifications">pipelining service mesh specifications</Link>.
