<p>
    <code>mesheryctl</code> runs on Intel and M1 Apple systems. Use your favorite package manager like Brew to install <code>mesheryctl</code> on your macOS. Install Meshery CLI with: <pre><code>brew install mesheryctl</code></pre>
</p>
<p>
    Keep your Meshery deployment updated with: <pre><code>mesheryctl system update</code></pre>
</p>
