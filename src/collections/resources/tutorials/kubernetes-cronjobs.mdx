<h3>Introduction</h3>
<p>In this tutorial, we explore Kubernetes CronJobs using <a href="https://play.meshery.io/">Meshery Playground</a>. CronJobs run Jobs on a scheduled basis, similar to cron tasks in Unix systems.</p>

<h3>Prerequisites</h3>
<ul>
  <li>Basic understanding of Kubernetes Jobs and scheduling concepts.</li>
  <li>Access to <a href="https://play.meshery.io/">Meshery Playground</a>.</li>
</ul>

<h3>Objective</h3>
<p>Learn how to create and manage <strong>Kubernetes CronJobs</strong> to schedule recurring tasks within a cluster.</p>

<h3>Steps</h3>

<h4>Access Meshery Playground</h4>
<ol>
  <li>Log in to <a href="https://play.meshery.io">Meshery Playground</a>.</li>
  <li>Click <strong>Explore</strong> to navigate to Kanvas.</li>
</ol>

<h4>Create a CronJob</h4>
<ol>
  <li>Rename the design to a name of your choice.</li>
  <li>Search for <strong>CronJob</strong> in Components and drag it onto the canvas.</li>
  <li>Configure: schedule <code>* * * * *</code>, image <code>busybox:latest</code>, command <code>/bin/sh -c "echo Hello from CronJob"</code>, label <code>app: my-cronjob</code>.</li>
</ol>

<h4>Deploy and Observe</h4>
<ol>
  <li>Click <strong>Validate</strong>, then <strong>Deploy</strong>.</li>
  <li>Click <strong>Visualize</strong> and filter by your label to see the CronJob and its spawned Jobs.</li>
  <li>Select a completed Pod and click <strong>Stream Container Logs</strong> to verify the output.</li>
</ol>

<h3>Conclusion</h3>
<p>You have successfully created and observed a Kubernetes CronJob using Meshery Playground. Continue exploring at <a href="https://play.meshery.io/">Meshery Playground</a>.</p>
