<h3>Introduction</h3>
<p>In this tutorial, we explore Kubernetes Services using <a href="https://play.meshery.io/">Meshery Playground</a>. Services provide stable network endpoints for Pods.</p>

<h3>Prerequisites</h3>
<ul>
  <li>Basic understanding of Kubernetes Pods and Deployments.</li>
  <li>Access to <a href="https://play.meshery.io/">Meshery Playground</a>.</li>
</ul>

<h3>Objective</h3>
<p>Learn how to create and configure <strong>Kubernetes Services</strong> to expose applications inside a cluster.</p>

<h3>Steps</h3>

<h4>Access Meshery Playground</h4>
<ol>
  <li>Log in to <a href="https://play.meshery.io">Meshery Playground</a>.</li>
  <li>Click <strong>Explore</strong> to navigate to Kanvas.</li>
</ol>

<h4>Create a Deployment</h4>
<ol>
  <li>Rename the design to a name of your choice.</li>
  <li>Search for <strong>Deployment</strong> in Components and drag it onto the canvas.</li>
  <li>Configure: name <code>web-deployment</code>, replicas <code>2</code>, image <code>nginx:latest</code>, label <code>app: web</code>.</li>
</ol>

<h4>Create a ClusterIP Service</h4>
<ol>
  <li>Search for <strong>Service</strong> and drag it onto the canvas.</li>
  <li>Configure: name <code>web-clusterip</code>, type <code>ClusterIP</code>, port <code>80</code>, targetPort <code>80</code>, selector <code>app: web</code>.</li>
  <li>Connect the Service to the Deployment using the relationship arrow.</li>
</ol>

<h4>Create a NodePort Service</h4>
<ol>
  <li>Drag another <strong>Service</strong> onto the canvas.</li>
  <li>Configure: name <code>web-nodeport</code>, type <code>NodePort</code>, port <code>80</code>, targetPort <code>80</code>, selector <code>app: web</code>.</li>
</ol>

<h4>Deploy and Visualize</h4>
<ol>
  <li>Click <strong>Validate</strong>, then <strong>Deploy</strong>.</li>
  <li>Click <strong>Visualize</strong> and filter by <code>app: web</code> to inspect resources.</li>
</ol>

<h3>Conclusion</h3>
<p>You have successfully created Kubernetes Services using Meshery Playground. Continue exploring at <a href="https://play.meshery.io/">Meshery Playground</a>.</p>
