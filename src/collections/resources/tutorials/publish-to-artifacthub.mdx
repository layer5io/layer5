<h3>Introduction</h3>
<p>In this tutorial, we learn how to publish Meshery Designs to <a href="https://artifacthub.io/">ArtifactHub</a>, a platform for finding and sharing cloud native packages and configurations.</p>

<h3>Prerequisites</h3>
<ul>
  <li>A Meshery design ready to publish.</li>
  <li>A GitHub account and a public repository.</li>
  <li>An <a href="https://artifacthub.io/">ArtifactHub</a> account.</li>
</ul>

<h3>Objective</h3>
<p>Learn how to <strong>export a Meshery Design</strong> and <strong>publish it to ArtifactHub</strong> so others can discover and reuse your infrastructure patterns.</p>

<h3>Steps</h3>

<h4>Export Your Meshery Design</h4>
<ol>
  <li>Log in to <a href="https://play.meshery.io">Meshery Playground</a> and open your design in Kanvas.</li>
  <li>Click the <strong>Export</strong> button in the toolbar.</li>
  <li>Save the exported <code>.yaml</code> file to your local machine.</li>
</ol>

<h4>Set Up a GitHub Repository</h4>
<ol>
  <li>Create a new public GitHub repository (e.g., <code>my-meshery-designs</code>).</li>
  <li>Add your exported <code>.yaml</code> design file to the repository.</li>
  <li>Create an <code>artifacthub-repo.yml</code> file in the repository root.</li>
  <li>Commit and push both files to the repository.</li>
</ol>

<h4>Register Your Repository on ArtifactHub</h4>
<ol>
  <li>Log in to <a href="https://artifacthub.io/">ArtifactHub</a>.</li>
  <li>Click your profile icon and select <strong>Control Panel</strong>.</li>
  <li>Click <strong>Add repository</strong>, enter a display name and the GitHub URL to your designs directory.</li>
  <li>Click <strong>Add</strong>. ArtifactHub will index your designs within a few minutes.</li>
</ol>

<h3>Conclusion</h3>
<p>Your Meshery Design is now published on ArtifactHub and discoverable by the cloud native community. Continue sharing at <a href="https://artifacthub.io/">ArtifactHub</a>.</p>
