<p><a href="https://ccoss.org/">CCOSS</a> is an Open Source Software Contributor Summit that focuses on promoting and increasing the contributions from people and organizations to Open Source in Latin America. This event will happen online from October 4th to 9th.</p>

<p><a href="https://layer5.io/service-mesh-management/meshery">Meshery</a> will be participating as a project in a <a href="https://ccoss.org/sessions/2021/taller-contribucion-meshery/">workshop</a> where attendees will contribute to Documentation, Tests, and more. It’s a good opportunity to learn how to contribute to Open Source supported by the community. To participate you can register <a href="https://ccoss.org/sessions/2021/taller-contribucion-meshery/">here</a>.</p>
