### What is the use of this CTA:
The **CardOutline** component is used to display a Card with title, description,image, link

### How to use the CTA:

```
<CardOutline
        link="/careers"
        img={Jobs_Icon}
        alt="Job"
        title="Careers"
        content="I'm interested in career opportunities at Layer5"
      />
```
