[Layer5.io](https://layer5.io) is a community-maintained repository of information pertaining to the technology ecosystem surrounding service meshes, API gateways, edge proxies, ingress and egress controllers - - microservice management in cloud native environments. 

# Projects

## [Meshery](https://layer5.io/meshery)
Meshery is a multi-service mesh management plane offering lifecycle management of more types of service meshes than any other tool available today. Meshery facilitates adopting, configuring, operating and managing performance of different service meshes and incorporates the collection and display of metrics from applications running on top of any service mesh. 

## [Service Mesh Landscape](https://layer5.io/landscape)
The [service mesh landscape](https://layer5.io/landscape) is a comprehensive collection of prominent service mesh projects and offerings laid out in contrast to one another.

## [Service Mesh Benchmark Specification](https://layer5.io/performance)
The [Service Mesh Benchmark Specification](https://layer5.io/performance) is a common format for describing and capturing performance benchmark test environmental configuration, mesh configuration and load test configuration.
